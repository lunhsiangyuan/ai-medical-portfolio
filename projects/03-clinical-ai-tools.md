# 🏥 臨床 AI 工具 — 每天在用的系統

## 概述

這些不是概念驗證或 demo，是我每天在臨床工作中實際使用的 AI 工具。

## 工具清單

### 1. OPD Note 自動化
門診截圖 → AI OCR 辨識 → 結構化 Google Document（雙欄 + 雙倍行距格式）

```
截圖上傳 → Gemini OCR → 計畫擷取 → Google Docs 格式化
                                         │
                                         ▼
                                    雙欄排版文件
                                    （直接可用）
```

### 2. PubMed 智能搜尋
自然語言描述研究問題 → 自動生成 MeSH 搜尋策略 → 批次下載 → 結構化摘要

### 3. 深度同儕審查系統
四維度系統性論文評估：
- 方法學嚴謹性
- 統計正確性
- 原創性評估
- 臨床意義分析

### 4. 臨床指引即時查詢
整合 NCCN / EAU 最新指引，對話式查詢特定臨床情境的建議。

### 5. 醫學英語訓練
- AI 即時口語糾正
- ElevenLabs TTS 發音示範
- 臨床情境對話練習
- FSI 方法論式 drill 訓練
