# 🎓 Lecture Producer — AI 教學自動化

## 概述

一套端到端的教學內容生產系統，從主題到完整教學影片全自動化。

## 工作流程

```
主題輸入
    │
    ▼
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  結構化講稿   │────▶│  HTML 投影片  │────▶│  PPTX 轉換   │
│  (Markdown)   │     │  (逐頁設計)   │     │  (pptxgenjs)  │
└──────────────┘     └──────────────┘     └──────┬───────┘
                                                  │
┌──────────────┐     ┌──────────────┐             │
│  完整影片     │◀────│  AI 語音旁白  │◀────────────┘
│  (Remotion)   │     │  (TTS)       │
└──────────────┘     └──────────────┘
```

## 實際案例

### Metastatic Bladder Cancer 2025
- **產出**：25 張投影片 + 40 分鐘英文旁白影片
- **時間**：從零到完成約 2 小時
- **品質**：包含最新臨床試驗數據（EV-302, CheckMate-901）
- **格式**：5 個臨床問題互動架構

## 能力

- 支援中英文投影片和旁白
- 自動嵌入數據表格和流程圖
- 可自訂投影片風格和品牌
- 輸出格式：PPTX / HTML / MP4
