# 🦞 OpenClaw — 多 Agent AI 管理平台

## 概述

OpenClaw（小龍蝦）是自建的 AI Agent 管理平台，部署於 Mac Mini 24/7 運行。透過統一的 Gateway 架構，管理多個 AI Agent 的生命週期、模型切換、通道綁定與記憶系統。

## 核心能力

### 多模型管理
- 支援 OpenAI (GPT-5.4)、Google Gemini、Claude (Antigravity)、本地 Ollama
- 4 層 Fallback 鏈：品質遞減、可用性遞增
- OAuth 自動刷新機制

### Agent 生命週期
- 完整 CRUD：建立、更新、刪除、狀態監控
- Agent Factory（蝦蝦工廠）：一條龍部署流程
- ACP/ACPX 協議層通訊

### 多通道派發
- 統一綁定機制：一個 Agent 可服務多個通道
- 支援 Telegram、LINE、Signal
- 動態路由：根據來源自動選擇 Agent

### 記憶架構
- 雙層設計：高階記憶（跨 session）+ 任務記憶（per session）
- 向量搜尋整合（LanceDB）
- 從「單次回答機器」進化為「複利累積系統」

## 設計哲學

- **可塑性** — 每個 Agent 獨立配置模型、workspace、權限
- **優雅降級** — 網路故障時自動切換至本地 Ollama
- **記憶複利** — 主動設計 memory protocol，讓 Agent 持續累積知識
- **多通道中立** — 綁定機制與平台無關，新增通道零成本
