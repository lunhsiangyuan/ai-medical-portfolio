# 🤖 Personal AI Infrastructure (PAI)

## 概述

PAI 是我從零建構的個人 AI 基礎設施，以 Claude Code 為核心引擎，整合 60+ 個客製化 Skills 和多種外部工具，形成一個完整的 AI 開發 + 部署生態系。
